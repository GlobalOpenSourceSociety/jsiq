### How do you find angular CLI version?

Angular CLI provides it's installed version using below different ways using ng command,

```bash
ng v
ng version
ng -v
ng --version
```

and the output would be as below,

```bash
Angular CLI: 1.6.3
Node: 8.11.3
OS: darwin x64
Angular:
...
```
